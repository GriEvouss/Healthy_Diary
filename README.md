# Дневник Здоровья - Клиент-серверное приложение

Курсовая работа по дисциплине "Архитектура клиент-серверных приложений"

## Описание проекта
Веб-приложение для мониторинга состояния здоровья и ведения медицинских записей.

## Функциональность
- Добавление и отслеживание симптомов
- Учёт приёма лекарств
- Просмотр истории записей
- Адаптивный веб-интерфейс

## Архитектура
- **Клиент**: HTML, CSS, JavaScript (Bootstrap 5)
- **Сервер**: Node.js + Express
- **База данных**: PostgreSQL
- **Контейнеризация**: Docker + Docker Compose
